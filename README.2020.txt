Source code for DRMM from:

https://github.com/faneshion/DRMM

It is easy to run this code in CSIL and just follow commands in build.sh.

Basically three steps to run the code.

1. Unzip the data. (tar jxf *.bz2)
2. Compile
3. Run

After running this code in a CSIL machine, 
a 5-fold cross validation result is produced, presenting training loss and testing loss.
